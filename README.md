# YelpCamp
This is my version of YelpCamp, the megaproject of the very famous Colt Steele Web Developer BootCamp. This is a modified version of the actual project, fixing some things and adding some features missing in the original.

This repositry does not contain the environment variables file as that contains private information. If you plan on downloading the code and running it locally make sure to add your own. Also run the 'npm install' command to install all the dependencies as the node modules folder is to big to upload on github.

The deployed website - https://adviks-yelpcamp.herokuapp.com/
